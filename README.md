# movie-site-open-research-notes
本仓库用于对互联网上已公开存在的影视类网站进行研究笔记式整理，围绕电影与电视剧相关站点的基础公开资料进行系统记录。通过统一整理框架，对站点名称、访问地址、站点定位说明及基础分类信息进行归纳，方便长期保存与反复查阅。本项目不涉及任何影视资源的存储、播放、下载或解析，仅针对公开网络环境中可获取的信息进行研究性整理。所有内容均来源于公开渠道，整理过程保持客观、中立，仅作为资料整理与信息研究用途存在。
